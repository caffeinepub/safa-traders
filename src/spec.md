# Specification

## Summary
**Goal:** Add a quotation request form for companies to request quotes for dismantling services.

**Planned changes:**
- Create a quotation request form component with fields for company details, site information, and service requirements
- Add backend storage and retrieval functionality for quotation requests
- Integrate a "Request Quote" button in the landing page services section that opens the form

**User-visible outcome:** Companies can submit quotation requests for dismantling services directly from the landing page by clicking a "Request Quote" button, filling out a form with their site details, and receiving confirmation of submission.
